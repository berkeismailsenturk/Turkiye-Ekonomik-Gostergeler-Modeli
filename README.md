# Türkiye Ekonomik Göstergeleri: Büyüme ve Yatırım Analizi (R)

Bu proje, Türkiye ekonomisinin makroekonomik dinamiklerini R programlama dili kullanarak analiz eden bir çalışmadır. 

### 📈 Analiz İçeriği
- **GSYH Trend Analizi:** Zaman serisi üzerinden büyüme hızının incelenmesi.
- **Rezerv Yönetimi:** Altın rezervlerinin seyri.
- **Dış Ticaret ve Yatırım:** İhracat rakamları ile Doğrudan Yabancı Yatırımlar (FDI) arasındaki korelasyon.

### 🛠 Teknik Yetkinlikler
- **Veri Manipülasyonu:** `dplyr`, `tidyr`, `zoo`
- **Görselleştirme:** `ggplot2`, `gridExtra`
- **Ekonometrik Hazırlık:** Gecikmeli değişkenler (Lags), hareketli ortalamalar (Moving Averages).
